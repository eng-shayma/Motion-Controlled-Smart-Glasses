# Motion-Controlled Smart Glasses System using AI & IoT (Graduation Project)

## 📌 Project Overview
This repository contains my final **Graduation Project**: an advanced, motion-controlled smart glasses system designed to interact with and control smart home environments. 

By utilizing a camera to track head and hand movements combined with Artificial Intelligence, the system interprets human gestures to control various smart devices (such as lights, appliances, and smart plugs) seamlessly and intuitively.

---

## 🎯 Key Features & Functionalities
* **Gesture & Motion Control:** Enables users to control smart devices simply by moving their heads and hands.
* **Client-Server Architecture:** Designed as a distributed system where:
  * **Client (Smart Glasses):** Responsible for collecting the user's real-time movements and transmitting them.
  * **Server (Remote Computer):** Responsible for interpreting the movements using AI and executing commands to control the devices.
* **Broad Compatibility:** Fully compatible with a wide range of smart home devices and smart plugs.
* **Multi-Domain Applications:** Highly scalable for deployment in Smart Homes, Smart Cities, Healthcare, and Industrial environments.

---

## 🛠️ System Architecture & Tech Stack
* **Programming Language:** **Python** – Used for core system development, server-client communication, and logic.
* **Computer Vision:** **OpenCV Library** – Utilized for real-time image processing, motion tracking, and object detection.
* **Core Technologies:** * Camera (Data Collection)
  * Processing Unit & Communication Unit
  * Artificial Intelligence & Machine Learning (CNN)
  * Internet of Things (IoT)

---

## 📊 Evaluation & Results
The system was evaluated through a comprehensive user study, which demonstrated:
* **High Accuracy:** Precise interpretation of simple head and hand gestures.
* **Ease of Use:** Users were able to interact with and control smart devices quickly and effortlessly, making smart assistive technology more accessible.
